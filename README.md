# DevOps Practical - Shell Scripts Learning

Learning shell scripts for DevOps automation tasks.

## Requirements

- Linux/Unix OS
- Bash shell
- sudo access (for some scripts)

## How to Execute

1. Navigate to the shell_scripts directory:
```bash
cd shell_scripts/
```

2. Give execute permission to the script:
```bash
chmod 763 script_name.sh
```

3. Run the script:
```bash
./script_name.sh
```

## Scripts

- `automation_backup.sh` - Backup automation script
- `useradd.sh` - User management script
