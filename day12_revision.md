| Command | Purpose |
|--------|--------|
| `pwd` | Show current directory |
| `ls` | List files |
| `ls -l` | List files with details |
| `ls -a` | Show hidden files |
| `cd /home` | Change directory |
| `cd ..` | Move one directory up |
| `mkdir project` | Create directory |
| `rmdir project` | Remove empty directory |
| `touch file.txt` | Create empty file |
| `cat file.txt` | Display file content |
| `nano file.txt` | Edit file |
| `vim file.txt` | Edit file with vim |
| `cp file1.txt file2.txt` | Copy file |
| `mv file1.txt file2.txt` | Rename or move file |
| `rm file.txt` | Delete file |
| `rm -r folder` | Delete directory recursively |
| `echo "text" > file.txt` | Write text to file |
| `echo "text" >> file.txt` | Append text to file |
| `head file.txt` | Show first 10 lines |
| `head -n 5 file.txt` | Show first 5 lines |
| `tail file.txt` | Show last 10 lines |
| `tail -n 5 file.txt` | Show last 5 lines |
| `clear` | Clear terminal |
| `history` | Show command history |
| `whoami` | Show current user |
| `id` | Show user ID information |
| `sudo useradd devuser` | Create new user |
| `sudo userdel devuser` | Delete user |
| `sudo groupadd devgroup` | Create group |
| `sudo groupdel devgroup` | Delete group |
| `ps` | Show running processes |
| `ps aux` | Show all running processes |
| `top` | Monitor system processes |
| `htop` | Interactive process viewer |
| `kill 1234` | Kill process by PID |
| `df -h` | Show disk usage |
| `du -h` | Show directory size |
| `free -h` | Show memory usage |
| `uptime` | Show system uptime |
| `systemctl status ssh` | Check service status |
| `systemctl start ssh` | Start service |
| `systemctl stop ssh` | Stop service |
| `systemctl restart ssh` | Restart service |
| `systemctl enable ssh` | Enable service at boot |
| `journalctl` | View system logs |
| `journalctl -u ssh` | View logs for ssh service |
| `chmod +x script.sh` | Make file executable |
| `chmod 755 script.sh` | Set file permissions |
| `sudo chown user file.txt` | Change file owner |
| `sudo chown user:group file.txt` | Change owner and group |
